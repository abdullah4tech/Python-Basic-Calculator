# Calculator Project 🧮

This is a simple calculator program that performs addition, subtraction, multiplication, and division.

## Usage

1. Run the program.
2. Enter two numbers as prompted.
3. Choose an operation from the menu.
4. View the result.

## Options

- Addition ➕
- Subtraction ➖
- Multiplication ✖️
- Division ➗
